# EduCore - Enterprise Learning Management System

## High-Level Description

EduCore is a scalable enterprise-grade Learning Management System inspired by platforms such as Udemy and Coursera. The platform enables students to discover and enroll in courses, consume multimedia content, track learning progress, and interact with instructors. Instructors can create, manage, and publish educational content through a dedicated portal.

The project focuses on building a production-quality full-stack application with modern web technologies and clean architecture.

## Core Features (Prioritized)

### P0 - Mandatory Features

1. User authentication and authorization.
2. Student dashboard.
3. Course catalog and search functionality.
4. Course details page.
5. Video/media player interface.
6. Instructor portal for course creation.
7. Asset uploading for lectures and thumbnails.
8. Student enrollment management.
9. Learning progress tracking.
